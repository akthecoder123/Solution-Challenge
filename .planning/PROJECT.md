# Smart Volunteer Coordination System

## What This Is
A data-driven volunteer coordination application for social impact. It matches volunteers to NGO tasks using an AI-driven approach (pgvector embeddings, geospatial matching).

## Core Value
Efficiently allocating volunteers to critical social needs using AI and optimizing resources for NGOs.

## Target Audience
- **NGO Coordinators:** To post tasks and manage volunteers.
- **Volunteers:** To find and sign up for opportunities that fit their skills and location.
- **Funders / Admins:** To view impact dashboards.
